# Aviation Parts Classifier

Ce projet vise à classifier différentes parties d'un avion (ex : aile, cockpit, moteur, queue, train d'atterrissage) à partir d'images, en utilisant le transfer learning avec un modèle de deep learning pré-entraîné.

## Stack

- Python
- PyTorch
- Streamlit

## Objectif

- Charger un dataset d'images de pièces / parties d'avion
- Entraîner un modèle de classification simple
- Déployer une petite interface web pour tester des images et afficher la prédiction

## Classes retenues
- cockpit
- engine
- tail
- wings
- wheels
